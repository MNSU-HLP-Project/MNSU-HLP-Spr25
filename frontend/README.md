# Frontend Documentation

## General Information

This frontend is built with React, Vite, and Tailwind. There exists a general file structure of the different roles componenets are housed in their respective folder under dashboard. So admin components would be under /src/dashboard/admin/. There are some utility functions in utils and the HLP information with lookfors and colors are stored under assets.

## Key Files

In the utils folder, axios.js handles the setup for the api requests, it sets the url and configures the requests. App.jsx handles all of the routing for the different components and future components can be routed through there.
